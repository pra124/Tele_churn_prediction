About Model

It handles both numerical and categorical data with proper encoding and scaling. The model is deployed using Flask, allowing real-time churn prediction through a web interface.

About the Model

The Churn Prediction System uses Gradient Boosting, an ensemble technique combining multiple decision trees to form a strong predictive model.

  • Training Accuracy – 85.5%
  • Testing Accuracy – 80.5%

It reduces overfitting and handles both categorical + numerical data effectively for telecom churn prediction.
